# 📊 Full-Stack Expense Tracker

A production-ready, full-stack web application designed to help users track their daily expenses, manage their finances, and securely store their data in the cloud. 

**[Live Demo]([https://your-vercel-link.vercel.app/](https://expense-tracker-green-omega.vercel.app/))** ---

## 🚀 Features

* **Secure User Authentication:** Complete account management system with secure sign-ups, encrypted passwords, and user login/logout functionality.
* **Expense Management (CRUD):** Seamlessly create, read, update, and delete individual financial records.
* **Persistent Cloud Storage:** Fully integrated PostgreSQL database hosted on Neon for permanent, safe data storage.
* **Serverless Architecture:** Deployed on Vercel's edge network, demonstrating a modern, serverless hosting environment using Python and WSGI.
* **Administrative Dashboard:** Built-in Django GUI to view database tables, manage user permissions, and edit records.
* **Static File Management:** Production-ready asset pipeline utilizing WhiteNoise to efficiently serve CSS and Bootstrap styling directly from the cloud.

## 🛠️ Tech Stack

* **Backend:** Python 3, Django
* **Database:** PostgreSQL (Neon.tech)
* **Frontend:** HTML5, CSS3, Bootstrap 
* **Deployment:** Vercel (Serverless), WhiteNoise (Static Assets)
